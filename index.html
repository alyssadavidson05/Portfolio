Projects

Driver Safety System

This project involved designing and implementing an innovative Driver Safety System using Python and a graphical user interface (GUI). The system was developed to enhance vehicle safety by monitoring and adjusting key elements to prevent accidents and respond to emergencies effectively.

Key features included:

  • Automatic seat and steering wheel adjustments to ensure proper airbag alignment.
  • Hand placement monitoring using pressure sensors, issuing alerts when hands were not on the steering wheel.
  • Drowsiness detection through sensors, with warnings issued for signs of fatigue.
  • Health monitoring to track the driver’s heart rate and automatically contact emergency services if the driver became unresponsive.

As the team leader, I brainstormed and conceptualized features, delegated tasks, developed the GUI, and integrated components into a cohesive system that was rigorously tested for accuracy and reliability.
